# Why Files Aren't Enough

## The Problem

Reading a raw file for every query is unbearably slow.

## Challenge

Implement the smallest program that demonstrates 'Files Aren't Enough' before reading the solution.

## Exercise

Write a minimal implementation of 'Files Aren't Enough', then measure where it breaks.
