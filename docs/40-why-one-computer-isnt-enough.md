# Why One Computer Isn't Enough

## The Problem

One machine can no longer hold all the data or traffic.

## Challenge

Implement the smallest program that demonstrates 'One Computer Isn't Enough' before reading the solution.

## Exercise

Write a minimal implementation of 'One Computer Isn't Enough', then measure where it breaks.

---

## Engineering Notes

This discovery exists because the previous approach failed under a real constraint. The lesson is not 'One Computer Isn't Enough' as a fact, but as a response to pressure.
