# Why Locks Exist

## The Problem

Two threads change the same value and corrupt it.

## Challenge

Implement the smallest program that demonstrates 'Locks' before reading the solution.

## Exercise

Write a minimal implementation of 'Locks', then measure where it breaks.

## Engineering Notes

This discovery exists because the previous approach failed under a real constraint. The lesson is not 'Locks' as a fact, but as a response to pressure.
