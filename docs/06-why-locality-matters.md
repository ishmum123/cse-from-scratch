# Why Locality Matters

## The Problem

The machine reads one byte, then another far away, and wastes most of its time.

## Challenge

Implement the smallest program that demonstrates 'Locality Matters' before reading the solution.

## Exercise

Write a minimal implementation of 'Locality Matters', then measure where it breaks.

---

## Engineering Notes

This discovery exists because the previous approach failed under a real constraint. The lesson is not 'Locality Matters' as a fact, but as a response to pressure.
