# Why Numbers Need Memory

## The Problem

You can count, but every new number erases the last one.

## Challenge

Implement the smallest program that demonstrates 'Numbers Need Memory' before reading the solution.

## Exercise

Write a minimal implementation of 'Numbers Need Memory', then measure where it breaks.
