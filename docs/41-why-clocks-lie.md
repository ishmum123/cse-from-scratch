# Why Clocks Lie

## The Problem

Two threads change the same value and corrupt it.

## Challenge

Implement the smallest program that demonstrates 'Clocks Lie' before reading the solution.

## Exercise

Write a minimal implementation of 'Clocks Lie', then measure where it breaks.

---

## Engineering Notes

This discovery exists because the previous approach failed under a real constraint. The lesson is not 'Clocks Lie' as a fact, but as a response to pressure.
