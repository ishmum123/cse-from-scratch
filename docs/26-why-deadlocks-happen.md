# Why Deadlocks Happen

## The Problem

Two threads change the same value and corrupt it.

## Challenge

Implement the smallest program that demonstrates 'Deadlocks Happen' before reading the solution.

## Exercise

Write a minimal implementation of 'Deadlocks Happen', then measure where it breaks.

## Engineering Notes

This discovery exists because the previous approach failed under a real constraint. The lesson is not 'Deadlocks Happen' as a fact, but as a response to pressure.
